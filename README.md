## 📱 주요 기능

1. **지도 기반 농구장 탐색**
   - 지역별 농구장 위치 확인
   - 접근성 정보 제공

2. **맞춤형 경기 참여**
   - 위치/날짜/시간 필터링
   - 경기 신청 및 관리

3. **경기 개설 및 관리**
   - 장소, 시간, 참가자 정보 설정
   - 접근성 요구사항 입력
